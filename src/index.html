API REST BIBLIOTECA 0.0.1-ALPHA

Consultas Simples

http://{host}:{host}/ /livros - retorna todas as pessoas /livro/:id - retorna o livro com um id informado

Consultas Complexas

2) http://{host}:{port}/livros/:id >> Retorna todos os livros em JSON

3) http://{host}:{port}/livro/:id >> Retorna um livro com determinado id informado

4) http://{host}:{port}/livro/:id/autor/ >> Retorna um autor de um determinado livro

5) http://{host}:{port}/livro/:id/editora/ >> Retorna um livro de uma determinada editora
